# TA1 Vectorization

## Description
This is the internal UIUC repo for the vectorization portion of DARPA CMASS project. 

## Requirements
This repo requires the following packages:
* rasterio
* shapely
* pyproj
* tqdm

## Script
vetorize.py is the script for running the vectorization step of the code.
